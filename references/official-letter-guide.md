# 공문(기안문) 작성 가이드

공공기관 공문서의 실무 규격 정리본이다. 근거 법령은
「행정업무의 운영 및 혁신에 관한 규정」(약칭 **행정업무규정**) 과
동 시행규칙, 행정안전부 행정업무운영 편람이다. 조문 표기는 이 문서
작성 시점 기준이며, 실제 적용 시에는 최신 개정본을 확인한다.

보고서 스타일과 **섞어 쓰면 안 되는 규격**이 많다. 보고서 규격은
`report-writing-guide.md` 를 참고한다.

---

## 1. 용지 및 여백

### 1-1. 용지

- A4 (210 × 297 mm) — 규정 제7조 제6항
- 세로(Portrait) 기본

### 1-2. 여백 (시행규칙 별표 4 기준)

| 방향    | mm   | HWPUNIT |
|---------|------|---------|
| 위쪽    | 20   | 5669    |
| 아래쪽  | 10   | 2835    |
| 왼쪽    | 20   | 5669    |
| 오른쪽  | 20   | 5669    |

보고서의 `상·하 15 mm` 와 가장 눈에 띄게 다르다. 공문은 상단 영역에
기관 로고/제목 블록을 확보하기 위해 상단을 더 크게 잡는다.

## 3. 항목 기호 체계 (8 단계)

공문 본문을 단락으로 쪼갤 때 **상위에서 하위로** 다음 순서로 번호를 매긴다
(시행규칙 제2조 제1항).

```
1단계:  1.   2.   3.   4.   …
2단계:  가.  나.  다.  라.  …
3단계:  1)   2)   3)   4)   …
4단계:  가)  나)  다)  라)  …
5단계:  (1)  (2)  (3)  (4)  …
6단계:  (가) (나) (다) (라) …
7단계:  ①   ②   ③   ④   …
8단계:  ㉮  ㉯  ㉰  ㉱  …
```

### 3-1. 띄어쓰기/들여쓰기 규칙

- **1.** 은 왼쪽 기본선에서 바로 시작 (앞 여백 없음).
- **가.** 부터는 한 단계 위 기호 위치에서 **오른쪽으로 2 타** 들여 쓴다.
- 기호와 내용 사이 **1 타** 띄움 (예: `1. 추진 배경`, `가. 세부 일정`).
- 항목이 두 줄 이상으로 넘어가면, 둘째 줄부터는 **항목 내용의 첫 글자에
  맞추어 정렬**한다.
- 동일 단계 항목이 하나뿐이면 기호를 붙이지 않는다.

### 3-2. 구성 예

```
수신  ○○○○ 교육지원청장(○○○과장)
(경유)
제목  ○○○○○○○○

1. ○○○○○○○○○○○○○○○
  가. ○○○○○○○○○○○○○
    1) ○○○○○○○○○○○○
      가) ○○○○○○○○○○
        (1) ○○○○○○○○○
          (가) ○○○○○○○○
2. ○○○○○○○○○○○○○○○○○○○○○○○○○○○○
   ○○○○○
```

> **"2 타"** 는 한글 1 자 또는 영문·숫자 2 자 너비에 해당한다.

## 4. 날짜·시간·금액 표기

### 4-1. 날짜 (규정 제7조 제5항)

- 숫자로 표기하되 "년/월/일" 글자는 생략하고 그 자리에 **온점(.)** 을 찍는다.
- 월·일은 **앞의 0 을 붙이지 않는다**.
- 요일은 괄호 안에 쓴다.

```
✕ 2026년 02월 13일 금요일
✓ 2026. 2. 13.(금)
```

### 4-2. 시간

- **24 시각제** 로 쓰고 "시 / 분" 글자는 생략, 사이에 **쌍점(:)** 을 둔다.

```
✕ 오후 3시 20분
✓ 15:20
```

### 4-3. 금액

- 아라비아 숫자로 쓰되, 필요 시 괄호 안에 한글로 병기한다.

```
✓ 금113,560원(금일십일만삼천오백육십원)
✕ 345천원
✓ 345,000원
```

## 5. "끝" 및 "붙임"

### 5-1. "끝" 표시

- 본문 마지막 글자 뒤로 **2 타** 띄우고 `끝.` 을 쓴다.
- 오른쪽 한계선에서 본문이 끝났다면 다음 줄 왼쪽 기본선에서 2 타 띄운 뒤 `끝.`.
- 본문이 **표** 로 끝나는 경우, 표 다음 줄 왼쪽 기본선에서 2 타 띄우고 `끝.`.
- 기재 영역이 중간에서 끝나면 다음 줄에 `이하 빈칸` 을 표기한다.

### 5-2. "붙임" 표시

- 본문 종료 줄 다음에 `붙임` 으로 시작하는 줄을 둔다.
- 첨부가 1 건이면 번호 `1` 을 붙이지 않는다.
- `붙임` 뒤에 **쌍점(:) 을 붙이지 않는다**.

```
…… (본문) …… 주시기 바랍니다.

붙임  2026학년도 운영 계획 1부.  끝.

──── (첨부 2 건 이상) ────

…… (본문) …… 주시기 바랍니다.

붙임  1. ○○○ 계획서 1부.
     2. ○○○ 서류   1부.  끝.
```

## 6. 관련 근거 표기

```
1. 관련: 교육정책과-1234(2026. 2. 13.)호
```

근거가 2 건 이상이면 하위 항목 기호로 나눈다.

```
1. 관련
  가. 교육부-1234(2026. 2. 1.)
  나. ○○교육청-5678(2026. 1. 10.), 「2026년 ○○○ 운영 계획」
```

## 7. 문장 부호 사용

| 부호       | 용법                           | 예시                     |
|------------|--------------------------------|--------------------------|
| 가운뎃점 · | 대등·밀접한 열거                | 융·복합, 시·군·구        |
| 쌍점 :     | 앞 글자에 붙이고 뒤로 1 타 띄움 | 원장: 홍길동             |
| 물결표 ∼   | 앞뒤 붙여 씀                    | 4. 23. ∼ 6. 15.         |
| 홑낫표 「 」 | 법령·문서 이름 인용            | 「초·중등교육법」         |
| 겹낫표 『 』 | 낫표 안의 낫표                  | 『○○』                   |

## 8. 공문 작성 4 원칙

| 원칙     | 핵심 요지                                                    |
|----------|-------------------------------------------------------------|
| 정확성   | 정확한 용어·문법, 사실 왜곡 없음                              |
| 용이성   | 수신자 입장에서 읽기 쉬운 어휘 선택                           |
| 성실성   | 성의 있고 진실하게, 적절한 경어                               |
| 경제성   | 1 건 1 매 원칙, 표준 기안문 서식 사용                          |

또한 「국어기본법」에 따라 **한글로 작성** 하고 필요 시 괄호 안에 한자·
외국어를 병기한다.

## 9. 코드 예시: 기본 공문 생성

`HwpxDocument.new()` 로 빈 문서를 만든 뒤 문단을 채우고 네임스페이스
정규화로 마무리한다.

```python
from hwpx.document import HwpxDocument
from hwpx_toolkit import normalize_namespaces_in_place

doc = HwpxDocument.new()

doc.add_paragraph("1. 관련: 교육정책과-1234(2026. 2. 1.)")
doc.add_paragraph(
    "2. 2026 학년도 디지털 전환 추진과 관련하여 다음과 같이 담당자 협의를 "
    "안내하오니, 대상자가 참석할 수 있도록 협조하여 주시기 바랍니다."
)
doc.add_paragraph("  가. 일시: 2026. 3. 5.(목) 15:00 ∼ 17:00")
doc.add_paragraph("  나. 장소: 본청 소회의실 8 (남부청사 4 층)")
doc.add_paragraph("  다. 대상: 관련 업무 담당자 20명")
doc.add_paragraph("  라. 내용: 디지털 전환 정책 방향 모색")
doc.add_paragraph("  마. 협조사항")
doc.add_paragraph("    1) 14:50 까지 참석자 등록")
doc.add_paragraph("    2) 청사 주차 공간이 협소하므로 대중교통 이용 권장")
doc.add_paragraph("")
doc.add_paragraph("붙임  2026학년도 디지털 전환 로드맵 1부.  끝.")

doc.save("letter.hwpx")
normalize_namespaces_in_place("letter.hwpx")
```
